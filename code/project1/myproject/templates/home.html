Project Details

This project is demonstrating an automated personalised Museum Mobile Guide. User interfaces affect the user experience tremendously and this is the reason why personalisation is introduced. There are two types of implementing personalisation explicitly and implicitly. Users tend to prefer implicit personalisation so that they are not interrupted during their navigation. The personalisation developed in this project is implicit aiming to achieve a result as accurate as an explicit personalisation can accomplish.

The literature review conducted enabled the identification of the different personas, their success criteria and their motivation. An initial prototype was created to collect data for the development of the supervise machine learning model that was integrated into the final version of the Demonstration. To develop the final version of the Demonstration, a first prototype was created to collect data to develop a supervised machine learning model that was integrated into the final version. The first prototype used a questionnaire designed according to the literature review’s results so that the interactive data collected by the system were mapped with the corresponding persona. The final Demonstration collects a variety of interactive data from the users as before and the supervised learning model process these data to predict the persona of the user and make interface modifications automatically to increase the user’s experience.

In the end, the project demonstrated a Museum Mobile Guide that is capable to collect interactive data, match each user with a persona and personalise the interface automatically even when the guide sometimes is for one-time use.
